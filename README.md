# SentinelAI - Adaptive Context-Aware API Security Firewall

## Overview
Intelligent middleware-based security system that protects 
API-driven applications from malicious traffic.

## Setup
```bash
npm install
npm start
```

## API Endpoints
- GET /security/dashboard
- GET /security/logs
- GET /security/metrics
- GET /security/status
- POST /test/simulate-attack

## Tech Stack
- Node.js + Express.js
- Custom Security Middleware
- Real-time threat detection
